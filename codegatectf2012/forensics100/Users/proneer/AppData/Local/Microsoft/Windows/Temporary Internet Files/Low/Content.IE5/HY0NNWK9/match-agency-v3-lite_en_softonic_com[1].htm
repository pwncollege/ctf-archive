Developer's description

“Advanced match making tool”

Match Agency v3 (Lite) is an advanced match making program designed for both individuals and match making agencies. This tool features a lot of options such as complex calculations of relevance between two people, analyzing based on a friend list, customizing the installation fields, searching by zip code or area and many more features beneficial for dating agents.

Match Agency v3 (Lite) users can manage multiple profiles with different galleries, input different profile information to make advanced match making searches, post private messages from one profile to another and have secret chats in public or secret zones, etc.

To maintain a higher level of control, Match Agency v3 (Lite) keeps all the pictures in a mysql database. This program also allows users to add funds to their accounts in order to purchase more profile slots and increase their rank.

In short, Match Agency v3 (Lite) is a program that can be helpful for both individuals and agencies that are looking for an advanced and complex match making tool.
