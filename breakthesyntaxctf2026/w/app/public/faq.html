What is W?

The most revolutionary image sharing platform in the world. Instead of posting PNGs, you upload a WASM painter that draws them for you. Less space used, less S3 costs.

How does it work?

When your post is displayed, the browser loads your WASM module and calls its exported fill function. Your code writes RGBA pixel data into linear memory, and the browser draws it onto a 320 x 240 canvas.

Browser calls fill(ptr, w, h, r, g, b)
↓
Your WASM writes w × h pixels (4 bytes each: R G B A) at ptr
↓
Browser reads memory → putImageData → canvas

What must my WASM module export?

  1. memory — a WebAssembly linear memory (at least 1 page = 64 KB)
  2. fill(ptr, w, h, r, g, b) — a function that writes pixel data

Parameters passed to fill:

  • ptr — byte offset in memory where pixel data starts (currently 0)
  • w, h — canvas dimensions (320, 240)
  • r, g, b — a suggested color (currently 255, 0, 0); you can ignore these

Your function must write exactly w × h × 4 bytes starting at ptr. Each pixel is 4 consecutive bytes: [R, G, B, A]. Set A to 255 for fully opaque.

Example: solid color fill (WAT)

This minimal example fills every pixel with the color passed in (r, g, b). Save it as fill.wat and compile with wat2wasm fill.wat -o fill.wasm (WABT toolkit). 

(module
  (memory (export "memory") 5)

  (func (export "fill")
    (param $ptr i32) (param $w i32) (param $h i32)
    (param $r i32) (param $g i32) (param $b i32)

    (local $i   i32)
    (local $len i32)
    (local $pixel i32)

    ;; pixel = r | (g << 8) | (b << 16) | (0xFF << 24)
    (local.set $pixel
      (i32.or
        (i32.or (local.get $r)
                (i32.shl (local.get $g) (i32.const 8)))
        (i32.or (i32.shl (local.get $b) (i32.const 16))
                (i32.shl (i32.const 255) (i32.const 24)))))

    ;; len = w * h
    (local.set $len (i32.mul (local.get $w) (local.get $h)))

    ;; for (i = 0; i < len; i++)
    (block $break (loop $loop
      (br_if $break (i32.ge_u (local.get $i) (local.get $len)))

      ;; store pixel at ptr + i * 4
      (i32.store
        (i32.add (local.get $ptr)
                 (i32.shl (local.get $i) (i32.const 2)))
        (local.get $pixel))

      (local.set $i (i32.add (local.get $i) (i32.const 1)))
      (br $loop)
    ))
  )
)

Example: horizontal gradient (WAT)

A more interesting painter that fades the color from full brightness on the left to black on the right. 

(module
  (memory (export "memory") 5)

  (func (export "fill")
    (param $ptr i32) (param $w i32) (param $h i32)
    (param $r i32) (param $g i32) (param $b i32)

    (local $x i32)
    (local $y i32)
    (local $offset i32)
    (local $fr i32)
    (local $fg i32)
    (local $fb i32)

    (local.set $y (i32.const 0))
    (block $ybreak (loop $yloop
      (br_if $ybreak (i32.ge_u (local.get $y) (local.get $h)))
      (local.set $x (i32.const 0))

      (block $xbreak (loop $xloop
        (br_if $xbreak (i32.ge_u (local.get $x) (local.get $w)))

        ;; scale color: component * (w - x) / w
        (local.set $fr (i32.div_u
          (i32.mul (local.get $r) (i32.sub (local.get $w) (local.get $x)))
          (local.get $w)))
        (local.set $fg (i32.div_u
          (i32.mul (local.get $g) (i32.sub (local.get $w) (local.get $x)))
          (local.get $w)))
        (local.set $fb (i32.div_u
          (i32.mul (local.get $b) (i32.sub (local.get $w) (local.get $x)))
          (local.get $w)))

        ;; offset = (y * w + x) * 4
        (local.set $offset
          (i32.add (local.get $ptr)
            (i32.shl
              (i32.add (i32.mul (local.get $y) (local.get $w)) (local.get $x))
              (i32.const 2))))

        (i32.store (local.get $offset)
          (i32.or
            (i32.or (local.get $fr)
                    (i32.shl (local.get $fg) (i32.const 8)))
            (i32.or (i32.shl (local.get $fb) (i32.const 16))
                    (i32.shl (i32.const 255) (i32.const 24)))))

        (local.set $x (i32.add (local.get $x) (i32.const 1)))
        (br $xloop)
      ))
      (local.set $y (i32.add (local.get $y) (i32.const 1)))
      (br $yloop)
      )
    )
  )
)
